# Simple ByteCase Notes Workflow

ByteCase Notes is designed around one simple loop:

```text
Write what you observed.
Index what supports it.
Reference indexed artifacts in the notes.
Check references.
Export the packet.
```

## 1. Start

Use the Start tab to see the workflow, continue notes, add artifacts, run Ready Check, or move to export review.

## 2. Case

Enter the case number, examiner, phase, and source description.

## 3. Notes

Write the examiner narrative. Reference indexed artifacts with the preferred format:

```text
[ART-001]
```

## 4. Artifacts

Add important supporting items to the artifact index. Examples include screenshots, exported files, reports, media, tool output, or other items that support the narrative notes.

Use **Save + Insert Ref** to save the artifact and immediately place the reference into the notes.

## 5. Check Refs

Run Check Refs before export. Review missing references, unused artifacts, duplicates, and non-standard reference formatting.

## 6. Export

Export JSON for continuity. TXT and DOCX outputs are optional report-friendly formats.
